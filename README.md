## Setup Instructions

### **1️⃣ Clone the Repository**
```sh

# Ecommerce-Admin-Panel

--Add, edit, and delete products
--User-friendly UI with React
--Secure API endpoints
--Database integration

  Tech Stack :

2)Backend: Node.js, Express.js
